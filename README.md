# Finlight Consulting

Static React/Vite website for Finlight Consulting with Calendly booking and file-based articles.

## Development

```bash
npm install
npm run dev
```

## Add An Article

1. Add any article image to `public/assets/`.
2. Add a new object to `src/articles.js`.
3. Commit and push to `master`.

GitHub Pages deploys automatically from `.github/workflows/pages.yml`.
